# Credit Scoring Web Application

Веб-приложение для оценки кредитоспособности клиентов, использующее модель машинного обучения (Random Forest).

## Структура проекта

- `web/` - Исходный код веб-приложения (FastAPI)
  - `app.py` - Основной файл приложения
  - `templates/` - HTML шаблоны
  - `static/` - CSS, JS, изображения и шрифты
- `models/` - Обученные модели и метаданные
- `data/` - Данные для обучения и обработки (необязательно для продакшена)
- `notebooks/` - Jupyter ноутбуки с анализом и обучением (для разработки)

## Локальный запуск

### 1. Подготовка окружения

Убедитесь, что у вас установлен Python 3.8 или выше.

Рекомендуется использовать виртуальное окружение:

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS/Linux
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

Находясь в корневой папке проекта, выполните:

```bash
python -m uvicorn web.app:app --reload
```

После запуска приложение будет доступно по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000)
